This assignment introduces learners to essential computer literacy skills, focusing on file management and productivity concepts that form the foundation of digital literacy. Designed for both Windows and Mac users, the assignment helps learners build confidence to navigate file systems, organize their work, and use basic tools efficiently. Learners will also develop foundational computer soft skills, including digital hygiene and adopting a troubleshooting mindset.

**Learning objectives**
1. Understand the importance of and leading practices for efficient file organization.
2. Find, open, move, and save files on your computer in an organized manner.
3. Identify different file types, their purposes and applications.
4. Utilize basic computer shortcuts to work more efficiently.

**Suitable courses:** Data analytics, AIS, AI, any class on technology
