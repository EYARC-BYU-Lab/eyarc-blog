EY has developed a leading-class analytics platform, EY Helix, for use by its Assurance practice to leverage analytics techniques and technologies to bring more improvements and quality to each audit. EY Helix is a proprietary tool and not available for the classroom. However, the EYARC has developed EY Helix General Ledger Analyzer for students, a simplified tool based on elements of EY Helix designed specifically for the classroom and using Tableau Desktop software. The tool helps students simulate real-world audit analytics procedures by using similar data, analyses and tools used by professional auditors.

Students can utilize this tool while completing a series of cases for Peach State University Hotel. Students do not need to know how to develop analytics and visualizations in Tableau to complete the case. A full user guide for the tool is provided for the students to help them navigate and understand the purpose of each analysis within the tool.

Revenue case: In this two-part case, students will perform select procedures using audit analytics to identify and assess risks, understand the revenue process and obtain substantive audit evidence using various analyses within the EY Helix General Ledger Analyzer for students tool.

**Suitable courses:** Auditing (external and internal); Fraud; Data analytics
