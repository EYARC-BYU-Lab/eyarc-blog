In this assignment, you will explore a new way of coding called vibe coding — a method that uses generative AI (GenAI) powered tools to help you generate, test and deploy code based on natural language prompts.
You don’t need to be a programmer to get started. Throughout this assignment, you’ll practice writing and debugging Excel formulas, building a basic HTML website, and using a tool, React, to create a simple game all with the help of AI. Along the way, you’ll learn how to fix errors, improve your prompts, better understand the code AI generates, and learn the limitations of AI-generated code.

**Learning objectives**
1. Understand the benefits and limitations of vibe coding.
2. Use vibe coding techniques to create functional Excel formulas.
3. Generate basic website layouts using HTML with the help of AI.
4. Recognize situations where vibe coding is effective and where traditional coding may be more reliable.

**Suitable courses:** Data analytics, AIS, AI
